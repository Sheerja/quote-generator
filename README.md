# 💬 Quote Generator

A responsive Quote Generator built using **HTML**, **CSS**, and **JavaScript**. The application fetches inspirational quotes from an API and allows users to browse through them using Previous and Next buttons.

## 🚀 Features

* Fetches quotes from an external API
* Displays quotes dynamically
* Previous and Next navigation
* Responsive design for desktop, tablet, and mobile
* Handles API loading efficiently
* Simple and user-friendly interface

## 🛠️ Technologies Used

* HTML5
* CSS3
* JavaScript (ES6)
* Fetch API
